# Hello-World
これはmarkdownで書かれます

markdownとはHTMLのいくつかの要素を使った文書をより簡単に書くためのマークアップ言語です。
ですからmarkdownでできることはHTMLでもできます。

たとえば \# 見出し と書くとこうなります。

# 見出し

\## 見出し と書くとこうです。

## 見出し

つまり\#は\<h1\>に変換され、\#\#は\<h2\>に変換されるわけです。

markdownでもいくつかの文字はエスケープする必要があります。それには #、=、-、*、などがあります。

*で囲んだ文字列は*　\<em\>に変換されます。
**で囲った文字列** は\<strong\>に変換されます。
